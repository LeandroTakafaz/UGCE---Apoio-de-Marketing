Observação:

Caso precise solicitar mais de um item, marque os itens desejados e preencha os campos de cada um. Cada instrução tem seu nome indicando para qual aba se refere.

  • Instruções:
    1. Eventos Digitais - Defina a mensagem principal que será transmitida na campanha, destacando os benefícios e diferencias do evento. Faça isso no Campo de Mensagem-Chave.

    2. Projetos - Canais de Atendimento devem conter os meios que serão utilizados para contato, como por exemplo: Whatsapp: (92) 00000-0000

    3. Landing Page - Deve enviar por email a apresentação do projeto ou material que deverá ser fonte principal de pesquisa para desdobrarmos o texto da LP.

    4. Landing Page Call-to-Action - Especifique qual a ação principal que os visitantes devem realizar na Landing Page, como preencher um formulário, fazer uma inscrição, agendar um atendimento, etc.

    5. Logo/Identidade Visual - Estilo de Design Desejado, deve ser descrito o estilo de design que você imagina para o Logotipo/Identidade Visual, como minimalista, elegante, lúdico, etc.
